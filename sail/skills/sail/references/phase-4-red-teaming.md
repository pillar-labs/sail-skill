## Phase 4: Agentic Red Teaming (Test) 

In the Test phase, agents undergo rigorous security assessments that simulate adversarial behaviors to uncover vulnerabilities, weaknesses, and risks. Unlike traditional AI testing focused on functionality and performance, agentic red teaming goes beyond standard validation to include goal hijacking, human-in-the-loop approval bypass, prompt injection across tool chains, privilege escalation through delegation, memory poisoning, inter-agent instruction smuggling, and cascading failure propagation.
