## Phase 6: Sandbox (Operate) 

During the Operate phase, agents run within isolated execution environments that limit blast radius. Sandboxing and zero-trust strategies separate agents from critical infrastructure and sensitive data while keeping them productive. Isolation applies to agent-to-agent communication as well: a compromised agent should not be able to pivot into another's scope.
